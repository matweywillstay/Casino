(ENG) Project description
A very basic Python casino. Initially, it will include standard modes (50/50 coin toss, guess a number from 5/10/1000), but roulette-type modes are planned for the future. The project exists purely for the experience; it offers no value and encourages no action. Casinos are evil, and the people who play them are fools.
